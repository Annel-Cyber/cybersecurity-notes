# Defensive Secuirty Basics - TryHackMe

## Objective
Learn what defensive secuirty is and what a SOC analyst does.

## What is Defensive Secuirty?
Defensive security is about protecting systems and stopping attacks before they cause damage.

## What a SOC Analyst Does
-Watch for susoicious activity

-Check alerts

-Find where the attack is coming from

-Investigate what is happening

-Stop the attack

## Steps I Learned
1. Check alerts from the system
2. Look for unusal activity or IP addresses
3. Investigate what the attacker is doing
4. Look at URL activity (What pages are being accessed)
5. Review actions taken by the system
6. Block the attacker's IP using firewall rules

## What I learned
-Attacks can be detected by watching system activity

-It is important to act fast

-Blocking the attacker can stop the attack

## How To Stay Safe
-Use a firewall

-Monitor activity regularly

-Block suspicious IP addresses
